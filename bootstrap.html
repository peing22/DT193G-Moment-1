Bootstrap

Bootstrap är ett populärt och kraftfullt ramverk som kan användas för att bygga responsiva och estetiskt tilltalande webbapplikationer med hjälp av HTML, CSS och JavaScript. Bootstrap erbjuder en samling fördefinierade stilregler, komponenter och layoutverktyg som gör det enkelt att skapa enhetliga och moderna användargränssnitt.

Jag tycker att Bootstrap erbjuder en bra dokumentation som gör det enkelt att förstå de grundläggande funktionerna i ramverket. Det är en stor fördel att Bootstrap har inbyggt stöd för responsiv design och att det finns många fördefinierade komponenter för exempelvis navigering, tabeller, formulär och knappar som kan användas för att snabbt skapa stilen för webbplatsen. Däremot upplever jag att valmöjligheterna när det gäller färg, typsnitt, storlekar osv. är begränsade i grundutförandet och jag hade uppskattat fler valmöjligheter utan att behöva skapa egna anpassningar.
