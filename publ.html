Publicering

Jag har valt GitHub Pages som plattform för publicering av min webbplats. GitHub Pages är en webbplatshostingtjänst som tillhandahålls av GitHub. GitHub Pages stöder en mängd olika programmeringsspråk, exempelvis HTML, CSS och JavaScript samt ramverk såsom React, Vue.js och Angular. GitHub Pages är i grunden gratis att använda och det går att skapa och publicera webbplatser utan någon direkt kostnad. Det inkluderar både privat och offentlig kod.

En fördel med GitHub Pages är att det är enkelt att använda då det går att publicera statiska webbplatser direkt från ett GitHub-repository. Webbplatsinnehållet hanteras dessutom med Git, vilket gör det enkelt att spåra ändringar och samarbeta med andra i olika projekt.

Något som kan betraktas som en nackdel är att GitHub Pages lämpar sig bäst för enkla, statiska webbplatser. Det är begränsat när det gäller att hantera databaser, serverlogik och andra avancerade funktioner.

Arbetssättet för att publicera filer till GitHub Pages: 1. Skapa ett GitHub-repository på GitHub där arbetsfilerna lagras. 2. Gå till inställningarna för aktuellt repository och klicka på " Pages". 3. Välj den branch som ska utgöra källan för webbplatsen. 4. Klicka på spara så publiceras webbplatsen dom blir tillgänglig via en URL i formatet https://.github.io/.
